Documentation

jquery.timepickr was created in a attempt to make the process of inputing time in a form as easy an natural as posssible.

Goals

  • Unobtrusive
  • Graceful degradation
  • Using maximum 2 clicks
  • Intuitive keyboard navigation
  • Be touch friendly (think iphone, tablet pc, etc..)

Dependencies

  • jquery.ui.js
  • jquery.utils.js
  • jquery.strings.js
  • jquery.arrayUtils.js

Demo

Basic usage

$(function(){
    $('#test-1').timepickr({handle: '#trigger-test'});
});
