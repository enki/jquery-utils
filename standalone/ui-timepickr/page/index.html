Indtroduction

jquery.timepickr was created in a attempt to make the process of inputing time in a form as easy an natural as posssible.

Goals

  • Be unobtrusive
  • Degrade gracefully
  • Use maximum 2 clicks
  • Intuitive keyboard navigation
  • Be touch friendly (think iphone, tablet pc, etc..)

Documentation

Basic usage

$(function(){
    $('#test-1').timepickr({
        handle: '#trigger-test'
        convention: 12 });
});
