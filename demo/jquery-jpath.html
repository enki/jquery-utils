Basic usage

// returns "3"
var obj = {a:{b: 'yay', c:[1,2,3]}}
$(obj).jpath('a c:eq(2)');

$.jpath.expr

exprdescription
:first Returns the first element of an object
:last Returns the last element of an object
:eq(N) Returns the Nth element of an array or the N proprety of an object
:is(str) Returns true if the object's value match str
:nth(N) Returns the Nth element of an array
