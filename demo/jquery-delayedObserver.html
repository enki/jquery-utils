Usage

$(document).ready(function(){
    $('input#demo-1').delayedObserver(function() {
        $('#demo-1-result').text('Typed: '+ $(this).val());
    }, 0.8);
});

Demo with select

Successive selections shorter than 3 seconds will not trigger the callback.
$('select#demo-2').delayedObserver(function() {
    $('#demo-2-result').text('Changed to: '+ $(this).val());
}, 0.8, {event: 'change'});
