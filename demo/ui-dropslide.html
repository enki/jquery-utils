Events

Event Default Description
click function(e, dropslide) { dropslide.hide(); } called when a li is clicked (before select)
select false function called on selection

Options

Options Default Description
tree false create tree from a JSON object
mode '2d' 2d or 3d mode
trigger 'mouseover' triggering event
top 6 margin-top (li)
left 0 margin-left (ol)
showDelay 0 delay before showing
hideDelay 0 delay before hiding

Methods

Method Arguments Description
showLevel id Show specified navigation level (id is the DOM position)
showNextLevel   Show the next level if present
show   Simple shorthand for showLevel(0)
hide   Hide all level
