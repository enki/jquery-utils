Basic usage

$(function(){
    $('#test-1').dropslide();
});

Options

Options Default Description
enabled true autoload elements with ui-dropslide class
tree false create tree from a JSON object
mode '2d' 2d or 3d mode
trigger 'mouseover' triggering event
top 6 margin-top (li)
left 0 margin-left (ol)
Events
click function callback function called when a li is clicked
